# prarabdhmisra.github.io

My personal portfolio site. Built with Jekyll + GitHub Pages (theme: `pages-themes/minimal`).

**Live:** https://prarabdhmisra.github.io

## How to edit

1. Edit `index.md` (all page content lives there, in plain Markdown).
2. To change the site title/description or theme, edit `_config.yml`.
3. Commit and push to `main`:
   ```powershell
   git add .
   git commit -m "content: update portfolio"
   git push
   ```
4. GitHub Pages rebuilds automatically. Changes are live in ~1 minute.

No local build tools required — GitHub builds the site on push.
